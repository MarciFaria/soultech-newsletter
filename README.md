# SoulTech Newsletter

#### Definição:

Projeto Integrador do bootcamp da Proz Talent Clould que consiste em uma página web no formato de 'Newsletter'. A ideia é um lugar onde as pessoas possam postar seus tutoriais, artigos e também acessar posts de notícias do mundo tech.
